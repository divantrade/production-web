## üìù Content Types

### Projects

**Purpose**: Showcase film and video projects

**Fields**:
- **Title**: Project name
- **Slug**: URL-friendly identifier (auto-generated)
- **Category**: Documentary, Commercial, Music Video
- **Description**: Brief project overview
- **Full Description**: Detailed project information
- **Featured Image**: Main project thumbnail
- **Gallery**: Additional project images
- **Video URL**: YouTube or Vimeo video link
- **Client**: Client name
- **Year**: Production year
- **Duration**: Video duration
- **Role**: Your role in the project
- **Featured**: Mark as featured project
- **Tags**: Searchable keywords

**Best Practices**:
- Use high-quality images (min 1920x1080)
- Keep descriptions concise but informative
- Always add relevant tags for better search
- Set featured projects strategically (max 6)

### Blog Posts

**Purpose**: Share industry insights and company news

**Fields**:
- **Title**: Post title
- **Slug**: URL identifier
- **Author**: Post author
- **Published Date**: Publication date
- **Excerpt**: Brief summary
- **Content**: Main post content (rich text)
- **Featured Image**: Main post image
- **Categories**: Post categories
- **Tags**: Searchable keywords
- **SEO Title**: Custom page title
- **SEO Description**: Meta description

### Team Members

**Purpose**: Showcase team members

**Fields**:
- **Name**: Full name
- **Position**: Job title
- **Bio**: Professional biography
- **Photo**: Professional headshot
- **Social Links**: LinkedIn, Twitter, etc.
- **Featured**: Show on main team page

### Testimonials

**Purpose**: Display client testimonials

**Fields**:
- **Client Name**: Testimonial author
- **Company**: Client company
- **Position**: Client's job title
- **Testimonial**: Quote content
- **Rating**: Star rating (1-5)
- **Project**: Related project (optional)
- **Photo**: Client photo (optional)

### Settings

**Purpose**: Global website settings

**Sections**:
- **Site Settings**: Logo, title, description
- **Contact Info**: Address, phone, email
- **Social Media**: Platform links
- **SEO Settings**: Default meta tags
- **Navigation**: Menu structure

## ‚úèÔ∏è Content Creation

### Creating New Content

1. **Click "Create" button** in the top bar
2. **Select content type** from the dropdown
3. **Fill in required fields** (marked with *)
4. **Add optional fields** as needed
5. **Preview content** using the preview pane
6. **Publish** when ready

### Rich Text Editor

The content editor supports:
- **Headings**: H2, H3, H4
- **Text formatting**: Bold, italic, underline
- **Lists**: Bulleted and numbered
- **Links**: Internal and external
- **Images**: Inline images with captions
- **Videos**: YouTube/Vimeo embeds
- **Code blocks**: For technical content
- **Quotes**: Blockquotes for testimonials

### Image Management

**Upload Guidelines**:
- **Format**: JPG or PNG preferred
- **Size**: Minimum 1920x1080 for featured images
- **File size**: Under 5MB for optimal performance
- **Alt text**: Always add descriptive alt text
- **Captions**: Add captions when relevant

**Image Optimization**:
- Images are automatically optimized
- Multiple sizes generated for responsive design
- WebP format used when supported

### Video Integration

**YouTube Videos**:
1. Copy the YouTube video URL
2. Paste in the "Video URL" field
3. System automatically extracts video ID
4. Thumbnail fetched automatically

**Vimeo Videos**:
1. Copy the Vimeo video URL
2. Paste in the "Video URL" field
3. System processes the embed code

## üîç SEO Optimization

### Page Titles

- Keep under 60 characters
- Include primary keyword
- Make descriptive and compelling
- Avoid keyword stuffing

### Meta Descriptions

- 150-160 characters optimal
- Include call-to-action
- Summarize page content
- Use active voice

### URL Slugs

- Use lowercase letters
- Separate words with hyphens
- Keep short and descriptive
- Avoid special characters

### Image SEO

- Always add alt text
- Use descriptive filenames
- Include keywords naturally
- Keep file sizes optimized

## üö® Common Issues and Solutions

### Images Not Displaying

**Cause**: Large file size or unsupported format
**Solution**: 
- Compress images before upload
- Use JPG, PNG, or WebP formats
- Check internet connection

### Video Not Playing

**Cause**: Invalid video URL or privacy settings
**Solution**:
- Verify video URL is correct
- Ensure video is public
- Check if video exists

### Content Not Updating

**Cause**: Caching or publishing issues
**Solution**:
- Clear browser cache
- Check if content is published
- Wait a few minutes for propagation

### Permission Errors

**Cause**: Insufficient user permissions
**Solution**:
- Contact administrator
- Verify user role and permissions
- Check if session expired
